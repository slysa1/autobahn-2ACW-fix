# Autobahn nach 56
## a hoi 4 submod for Kaiserreich

Goal of this mod is to merge the tech from Road to 56 with Kaiserreich.

A Redux version is developed in parallel.

## Diagnostics and Recovery Notes

This repo now also contains compatibility investigation notes for getting a larger Kaiserreich-centered mod stack working again on current HOI4 versions.

Key files:

- `diagnostics/kr_playset_recovery_roadmap.txt`
  Main execution roadmap for stabilizing the playset, rebuilding stale compatches, and reintroducing secondary mods in controlled batches.

- `diagnostics/ai_project_briefing.txt`
  Fast handoff briefing for another AI tool or contributor. Explains project goals, proven findings, current decisions, and the recommended next steps.

- `diagnostics/extracted_log_errors.md`
  Grouped error families pulled from the current HOI4 logs.

- `diagnostics/file_ownership.md`
  Effective file ownership map showing which mod currently wins specific high-risk files in the load order.

- `diagnostics/mod_source_map.md`
  Broader source map for overlapping files across the active mod stack.

- `diagnostics/consistency_audit.md`
  Additional compatibility notes and cross-checks gathered during investigation.

If you are helping with compatibility work, start with the roadmap and AI briefing before making changes.
